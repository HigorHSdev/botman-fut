# Botman - AI Agent Instance

⚽ Botman Fut

Bot de Telegram desenvolvido para automatizar o envio de informações sobre futebol, utilizando integração com banco de dados em nuvem e deploy contínuo.

🚀 Sobre o projeto

O Botman Fut é um bot criado para facilitar o acesso a informações relacionadas ao futebol diretamente no Telegram, de forma rápida e automatizada.

O projeto foi desenvolvido com foco em aprendizado prático de backend, integração de serviços e deploy em nuvem, garantindo que o bot funcione 24 horas por dia.

🧠 O que eu aprendi

Durante o desenvolvimento deste projeto, tive a oportunidade de aprender e aplicar:

Integração com a API do Telegram
Criação de bots com Node.js
Uso do Supabase como banco de dados em nuvem
Deploy e hospedagem com Render
Manipulação de variáveis de ambiente
Estruturação de projetos backend
Automação de processos
🛠️ Tecnologias utilizadas
- Node.js
- JavaScript
- Telegram Bot API
- Supabase
- Render
  <br/>
- ⚙️ Funcionalidades
- 🤖 Respostas automáticas via Telegram
- ⚽ Envio de informações relacionadas ao futebol
- ☁️ Armazenamento de dados em nuvem
- 🌐 Bot online 24/7
- 📡 Integração com APIs externas (se aplicável)
  <br/>
Este projeto segue a arquitetura de 3 camadas definida em `AGENTE.md`.
